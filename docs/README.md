# Documentation

This folder contains project documentation and user guides.

## Contents

### `data_quality_guide.md`
Comprehensive user guide for the data quality monitoring system.

**Topics Covered:**
- How to run the data quality analysis notebook
- Interpreting quality scores and categories
- Understanding alias resolution statistics
- How to add new country and material aliases
- Troubleshooting common data quality issues
- Using quality metrics in Power BI dashboards
- Best practices for maintaining data quality

**Target Audience:** Data engineers, analysts, and stakeholders who need to understand and maintain data quality in the pipeline.
